Ходовая часть робота (Adafruit Driver)

Nov 22, 2014 • OrelSokolov

Данный стенд сделан для быстрого понимания того как можно управлять моторами с помощью драйвера Adafruit.

Стенд включает в себя:

  • Arduino Uno
  • Adafruit Driver
  • 2 мотора с проводами
  • USB кабель

Тут используется Adafruit Motor Driver

#include <AFMotor.h>

AF_DCMotor motorL(1);
AF_DCMotor motorR(4);

void setup() {
}

void loop() {

  motorL.run(FORWARD);
  motorL.setSpeed(240);

  motorR.run(FORWARD);
  motorR.setSpeed(240);

}
