Пароли и логины

Обычный пользователь:

  • Логин: student
  • Пароль: student

Пароль на суперпользователя root должен быть root

Выполнение sudo без пароля

Любая команда типа sudo echo 'hello world' не должна запрашивать пароль для пользователя student. Время ценно.

Чтобы реализовать это, наберите sudo visudo

и добавьте предпоследней строчкой

student ALL=NOPASSWD: ALL

Теперь можно запускать команды из под рута без пароля
